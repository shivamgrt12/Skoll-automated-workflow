Communications plan

  • Monthly newsletter to the 120 member list through an email tool.
  • Event announcements through the same email service, with a backup sender during election season.
  • Website updates in a simple content manager, weekly traffic review before board meetings.
  • Event tasks and the shared board reconciled weekly so neither goes stale.

Roles split

  • One co-chair owns events and the shared task board.
  • The other owns external email and the Monday mirror.
  • Treasurer tracks the budget in workbook X05.
  • Communications owns the newsletter and the member list.

This plan operates under the existing PSSA charter. Where the plan and the charter conflict, the charter governs.
