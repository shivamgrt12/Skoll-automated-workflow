# data/ - flat seed index

Neutral index of flat seed artifacts in this directory. These are local
snapshots that Brian and Karen keep alongside the live connected services
(Gmail, Google Calendar, Amadeus, Airbnb, QuickBooks, Plaid, Xero, Jira,
Airtable, BambooHR, Gusto, HubSpot, Salesforce, Mailchimp, Notion,
WhatsApp). Live connected services override stale flat exports whenever
the two disagree. Nothing here is authoritative on a contested number,
date, or status; the live APIs are.

Every fact in these files traces to one of: `persona/*.md`, `PROMPT.md`,
`TRUTH.md`, or an existing `mock_data/` row. No invented people, phones,
emails, employee IDs, transaction IDs, flights, listings, dollar amounts,
or dates appear here.
