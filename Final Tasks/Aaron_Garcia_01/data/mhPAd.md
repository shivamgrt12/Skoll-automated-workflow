# Mykonos Taverna Standard Operating Procedures

## Food Safety

All staff must complete ServSafe certification within 30 days of hire.
Temperature logs recorded every 2 hours during service.
Walk-in cooler: 38F target, alarm at 42F.
Freezer: 0F target, alarm at 5F.

## Server Training

New servers shadow for 3 shifts before going solo.
Menu knowledge test required (80% minimum).
Wine list familiarity: know top 5 by-the-glass options.
Upselling targets: appetizer mention rate 90%, dessert 70%.

## Closing Procedures

Kitchen close: Deep clean stations, label all prep, lock walk-in.
FOH close: Run POS report, count cash drawer, lock safe.
Manager: Review labor cost vs revenue for the day.
Verify all doors locked, alarm set (code changes quarterly).
Last person out signs the closing log.
