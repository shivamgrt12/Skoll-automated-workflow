- Phones, emails, and addresses live only in the stored Contacts.
- Connected service accounts (what is connected) live in the stored Connected Accounts; how the assistant uses those tools lives in the connected tool list; do not duplicate descriptions.
- Strip stale catering quotes, expired permits, and past events from stored memory into a rolling note rather than letting them clutter the canonical sections.
