# Journeyman Electrician - Milwaukee Market Reference

Reference snapshot for the crew-capacity decision. Pulled into Greenhouse as the standing market posting context. Figures reflect Milwaukee-area IBEW Local 494 jurisdiction and union/non-union blended Q4 2026 data. **All figures are reference only.** Any actual posting and hire is held for Jae's explicit approval.

## Recruitment timeline

| Phase | Typical duration |
|---|---|
| Posting open | 2-3 weeks |
| Resume screen + first conversation | 1 week |
| Working interview / ride-along | 1 week |
| Reference + license verification | 1 week |
| Offer + acceptance + start date | 1-2 weeks |
| **Total: posting to first day on a truck** | **6-8 weeks** |

> A hire opened in early October 2026 would realistically start mid-November 2026 to early December 2026. The Bay View Historic Home kickoff on November 2 would not have the fourth body in the truck for the first two to four weeks.
