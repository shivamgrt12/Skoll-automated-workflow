### Category 6: Analytical Precision

**Vulnerability: HIGH**

#### Why This Persona Is Exposed

Trade compliance is **fundamentally an arithmetic profession**. Tariff rates, duty drawback percentages, USMCA regional value content, transaction values, and Section 301 exposure all compound from base figures into client-facing dollar amounts. The persona's own mantra — "in trade, a misread number costs real money" — is the explicit acknowledgment that this is a precision domain.

**Tariff and duty math:**

- Standard duty calculations: HTS rate × dutiable value = duty owed. A 2.5% misread vs a 25% misread on a $500,000 shipment is a $112,500 difference. Decimal-point errors are the canonical precision hazard.
- Section 301 ad valorem tariffs (currently 25% on many Chinese-origin lines) layer on top of base duty. Mis-stacking these produces five-figure errors.
- USMCA regional value content (RVC) calculations under the de minimis, net cost, and transaction value methods. Tracing every component back to North American origin and computing the RVC percentage is a multi-step arithmetic exercise where any single supplier-value error cascades.
- Duty drawback math: refund equals 99% of duties paid on inputs to exported goods, subject to time limitations and documentation requirements. The 99% factor and the time-window math are easy to get wrong.
- Anti-dumping and countervailing duty rates layered on Section 301 — three tariff regimes stacking on a single product. Off-by-one in stacking order changes the result.
